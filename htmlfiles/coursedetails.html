BASIC COURSE

Outcomes of this Course:

         1. Analyze the woven fabric parameters such as weave structure, etc.
         2. Create and develop jacquard designs manually and by using CAD tool.
         3. Calculate yarn requirement for the developed design.
         4. Prepare resign report with the tie up instructions and punch card instructions.

Course Duration:

         2 Months

Entry Eligibility Requirements:

         Class 10th

ADVANCED COURSE

Outcomes of this Course:

        1. Create and develop jacquard manually and by using CAD tool
        2. Draw the basic weaves as per the standard fabric structure
        3. Operate the CAD tool to develop the required jacquard design
        4. Calculate warp and weft yarn requirement for the developed design
        5. Develop sample on the loom for created jacquard design
        6. Create and plan the tie up method for the various jacquard designs
        7. Explain the working mechanism of the mechanical and electron

Course Duration:

        2 Months

Entry Eligibility Requirements:

        Class 10th
